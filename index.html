  2. Build Your Own World (BYOW)

    3. This is a project that I also completed in Summer 2022. This was my second favorite project. It requires us to design and implement a 2D tile-based world exploration engine. This project especially stood out to me because it was interactable. The resulting product is able to take any user inputted seed, and use it to produce a world that's unique to that seed. The player is then able to move an avatar around the map and traverse collecting coins. The players also have the option to load a previous save or create a new game. This project was the culmination of world generation algorithms, interactivity, and overall a lot of creative coding fun!
